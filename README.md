# AvxUInt
 AVX Accelerated BigUInt Arithmetic Implements

## Requirement
 .NET 8.0

## Usage
```csharp
BigUInt<N4> v1 = "74607431768211455";
BigUInt<N4> v2 = "3402823669209384634633746074";

BigUInt<N4> v3 = v1 + v2;
```
